# SimuladorFácil — Ferramentas Financeiras

Este projeto é um dashboard financeiro completo que inclui cotações de mercado em tempo real, simulador de investimentos e comparador de ativos.

## 🚀 Funcionalidades

- **📊 Mercados**: Acompanhamento de cotações de metais, energia e produtos agrícolas.
- **🏠 Simulador**: Ferramenta para simular rendimentos de investimentos ao longo do tempo.
- **📈 Comparador**: Comparação direta entre diferentes tipos de ativos financeiros.

## 🛠️ Tecnologias

- **React**: Biblioteca principal para a interface.
- **Vite**: Ferramenta de build rápida para desenvolvimento web moderno.
- **CSS-in-JS**: Estilização integrada diretamente no componente.

## 📦 Como rodar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/vulnexusai/Finacial-
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
